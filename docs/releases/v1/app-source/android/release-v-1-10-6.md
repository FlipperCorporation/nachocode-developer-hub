# Release: ver.1.10.6 (2026-04-29)

import { ThumbnailImage } from '@site/src/components/common/image/thumbnail-image';

<ThumbnailImage src='/img/docs/releases/release_note_android_detail.png'/>

> 🔔 **배포 일자:** 2026-04-29

nachocode Android App Source **1.10.6** 버전이 반영되었습니다.

이번 업데이트 **ver.1.10.6**에서는 커스텀 인디케이터 기능이 추가되고, 마케팅 푸시가 개선되었습니다.

## 개선 및 수정 사항

- **마케팅 푸시** : 일부 동작 오류 및 안정성이 개선되었습니다.

## 새로운 기능

- **이미지 인디케이터 지원** : 커스텀 인디케이터에 이미지(`gif`, `webp`, `lottie`)를 적용할 수 있는 기능이 추가되었습니다.
