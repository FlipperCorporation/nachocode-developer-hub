# Release: ver.1.9.2 (2026-02-11)

import { ThumbnailImage } from '@site/src/components/common/image/thumbnail-image';

> 🔔 **배포 일자:** 2026-02-11

nachocode iOS App Source **1.9.2** 버전이 반영되었습니다.

이번 업데이트 **ver.1.9.2**에서는 일부 기능의 마이너 버그가 수정되었습니다.

## 개선 및 수정 사항

- **OS 권한 후 팝업 버그 개선**: OS 권한 팝업 이후 네이티브 팝업이 표시되지 않던 문제를 수정했습니다.
- **User Agent 버그 개선**: User Agent가 정상적으로 표시되지 않던 문제를 수정했습니다.

## 업데이트 적용 방법
