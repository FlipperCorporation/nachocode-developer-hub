애플 (apple) - 연동하기

🚀 추가된 버전 : v1.4.0 v1.4.0
🔔 최신화 일자: 2025-09-30

참고

이 네임스페이스는 현재 iOS에서만 지원됩니다.

필수 선행 작업​

nachocode SDK로 애플 네이티브 기능을 사용하기 위해서는 아래 사항이 먼저 완료되어야 합니다.

1. Apple Developer에서 Identifier 설정​

  • Identifier에 Capabilities 중 [ Sign In with Apple ] 추가


2. nachocode 대시보드에서 토글 활성화​

  • [ 앱 설정 > 개발자 설정 > 애플 네이티브 설정 ]에서 [ 애플 로그인 사용 ] 토글을 활성화하여 앱 설정 완료


3. nachocode 대시보드에서 재빌드 수행​

  • [ 앱 빌드 > iOS 앱 빌드 ]에서 [ 새 버전 만들기 ] 버튼을 클릭하여 빌드

  • 애플 관련 네이티브 기능은 새로 빌드 된 버전의 앱부터 적용됩니다.
